 # Security Policy

## Reporting a Vulnerability

If you believe you have discovered a security vulnerability affecting KinkyFans, please do not report it through public GitHub Issues.

Instead, report security issues responsibly through the official contact channels available on:

https://kinkyfans.net

https://kinkystars.com

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Screenshots or proof of concept if available

## Responsible Disclosure

We ask security researchers to:

- Avoid accessing user data unnecessarily.
- Avoid disrupting platform availability.
- Give us reasonable time to investigate and resolve reported issues before public disclosure.

We appreciate responsible security research that helps improve the safety and reliability of the platform.

## Scope

This policy applies to public resources published within this repository.

## Thank You

Thank you for helping improve the security of KinkyFans.
